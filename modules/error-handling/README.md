# error-handling

## 역할

공통 예외 응답 처리 모듈이다.
서비스별 비즈니스 예외를 HTTP 오류 응답으로 일관되게 변환하는 기반을 제공한다.

## 제공 기능

- 공통 `ErrorCode` 계약
- 공통 예외 타입
- Spring MVC 기반 예외 응답 처리
- ProblemDetail 또는 공통 에러 포맷 기반 확장 지점

## 방향성

- 서비스별 `ErrorCode` 는 각 서비스 내부에 둔다.
- 공통 모듈은 예외 처리 메커니즘만 제공한다.
- 비즈니스 정책은 여기 넣지 않는다.

## 넣지 않을 것

- 서비스별 에러 코드 enum
- 인증/인가 정책
- 도메인별 예외 분기 로직

## 사용 대상

- `auth-service`
- `user-service`
- 이후 HTTP API를 노출하는 모든 서비스
